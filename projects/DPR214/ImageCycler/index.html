Generic Image Cycler

Below is presented a generic image cycler written in JS and jQuery. The cycler loads image data from this local json file and cycles through the images in order. However, there is also a random mode which shuffles the images and goes through them in a somewhat random order (without repeats). You can set the mode and delay between image changes below. Right now, the utility cycles through several images of my server lab through the years. Each image has a random URL associated with it.
